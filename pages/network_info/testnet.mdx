# Testnet

In order to connect to the Abstract testnet, you'll need to add a new network with the following details:

- Name: **Abstract Testnet**
- Description: **The public testnet for Abstract.**
- Chain ID: `11124`
- RPC URL: `https://api.testnet.abs.xyz`
- Explorer: `https://explorer.testnet.abs.xyz`
- Currency Symbol: `ETH`

To switch to Abstract's network, simply click on the button below once you have a wallet installed.
